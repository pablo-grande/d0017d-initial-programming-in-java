## Step 2 — Open in Codespaces

Once you have your own copy, open it in a fully configured coding environment in your browser:

1. Go to **your** repository on GitHub
2. Click the green **Code** button
3. Select the **Codespaces** tab
4. Click **Create codespace on main**

> The environment takes about two to three minutes to set up the first time.

---

## Step 3 — Run your first program

1. Open `Hello.java` in the file explorer on the left
2. Click the **▶ Run** button in the top right corner of the editor
3. Watch your program run in the terminal below

---

## Your First Java Program

```java
void main() {
    System.out.println("Hello, World!");
}
```

This is a complete Java program. No classes, no `public static void main(String[] args)` — just the code that matters. You will learn what those things mean later in the course, step by step.

You can also run any `.java` file directly from the terminal:

```
java Hello.java
```

## About This Course

This course uses **Java 25**, which introduced a simplified way to write programs so that beginners can focus on learning to think like a programmer — without boilerplate code that gets in the way at the start.
