## Your First Java Program

```java
void main() {
    System.out.println("Hello, World!");
}
```

This is a complete Java program. No classes, no `public static void main(String[] args)` — just the code that matters. You will learn what those things mean later in the course, step by step.

You can also run any `.java` file directly from the terminal:

```
java Hello.java
```

## About This Course

This course uses **Java 25**, which introduced a simplified way to write programs so that beginners can focus on learning to think like a programmer — without boilerplate code that gets in the way at the start.
